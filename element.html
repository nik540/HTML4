VSC Figma element атрибуты CSS практика flexbox grid

Элементы

Строчные элементы

< br > - перевод строки
< span > - выделенные слова
< strong > - акцентирование текста, жирный шрифт
< q > - короткая цитата, отображается в кавычках
< abbr > - аббревеатуры
< small > - менее важные слова
< a > - ссылка
< img > - изображение
< sub > - нижний индекс
< sup > - верхний индекс

Текстовые элементы

< p > - абзац
< ul > - неупорядоченный список
< ol > - упорядоченный список
< dl > - список описаний
< h1...h6 > - заголовок
< article > - статья
< section > - раздел
< blockquote > - длинные цитаты
< pre > - текст со всеми пробелами

Строчно-блочные элементы

< li > - пункты списка
< dt > - определение термина
< dd > - описание термина
< table > - таблицы
< tr > - строки таблицы
< td > - ячейки таблицы

Структурные элементы

< header > - первый элемент страницы
< nav > - список ссылок
< h1...h6 > - заголовки
< article > - основное содержимое
< footer > - подвал страницы
< main > - основное содержание
< aside > - раздел доп.информации
< section > - раздел

Тематические элементы

< output > - результат вычисления
< progress > - процес вычисления

Общие элементы

< div > - общий блочный элемент
< span > -общий строчный элемент

Таблицы

< table > - таблица
< tr > - добавить строк
< td > - добавить обычные ячейки
< th > - добавить заглавные ячейки
< thead > - шапка таблицы
< tbody > - тело таблицы
< tfoot > - подвал таблицы

Формы

< form > - блочный элемент для форм
< input > -создание интерактивных элементов
< textarea > -элемент формы для создания многострочного поля
< iframe > - встроенный фрейм
< datalist > - список подсказок
< option > - определяет отдельные пункты списка
< button > - создает кнопку

Спецсимволы

< &lt > - знак "меньше"
< &gt > - знак "больше"
< &quot > - двойная кавычка
< &amp > - амперсанд
< &nbsp > - неразрывный пробел
< &shy > - мягкий перенос

Переносы текста

< nbsp > - неразрывный пробел
< wbr > - желательный перенос
< shy > - желательный перенос, при переносе добавляет дефис
