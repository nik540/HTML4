VSC Figma element атрибуты CSS практика flexbox grid

grid

Варианты объявления grid-контейнера

grid
генерирует grid-контейнер уровня блок
inline-grid
генерирует grid-контейнер уровня строки

Определение явной сетки

grid-template-rows
количество строк
grid-template-colums
количество столбцов
grid-template
сокращенное свойства количества строк/столбцов
grid-template-areas
создает grid-области путем размещения именованных ячеек в грид-раскладке

Определение неявной сетки

grid-auto-columns
устанавливает размер неявных столбцов
grid-template-rows
устанавливает размер неявных строк
grid-auto-flow
управляет размещением неявных элементов

Выравнивание содержимого вдодль горизонтальной оси

justify-items
выравнивает содержимое вдоль оси строки

Выравнивание содержимого вдоль вертикальной оси

aling-items
выравнивает содержимое вдоль вертикальной оси

Выравнивание сетки вдоль горизонтальной оси

justify-content
выравнивает сетку вдоль горизонтальной оси

Выравнивание сетки вдоль вертикальной оси

aling-content
выравнивает сетку вдоль вертикальной оси
